Help

Timer Controls

Start begins the game timer. Pause freezes the countdown mid-round. Reset stops the timer and returns to round 1.

The timer counts down each round, sounding the siren when a round ends. After a short pause it automatically starts the next round.

Pause Next (operator/admin only) queues a pause between the current round and the next. Tap again to cancel. Useful for speeches or breaks without interrupting the current round.

Game duration (1–120 min) and number of rounds (1–20) can be changed from the main page while the timer is idle.

Siren blast length and pause between blasts can be adjusted in Advanced Settings (100–10,000 ms).

Upcoming Events

Shows the next 7 days of Hello Club bookings that have the timer: tag in their description.

Events auto-trigger the timer at their start time with the configured duration and rounds.

Greyed-out events have already been triggered. The timer won't fire the same event twice.

Events sync automatically every hour. Admins can force a sync with the refresh button.

Hello Club Setup

Add timer: to any Hello Club event description to have it auto-trigger the court timer. The siren sounds at the end of each round. By default, rounds repeat until the booking ends. Examples:

  • timer: — uses default round duration (set in Settings), continuous
  • timer: 15min — 15-minute rounds, continuous
  • timer: 15min 3rounds — 15-minute rounds, stops after 3
  • timer: 10min 5rounds — 10-minute rounds, stops after 5

The tag can appear anywhere in the description. Only events within the next 7 days are fetched.

Configure the API key and enable auto-trigger in Settings > Hello Club API.

QR Codes

The QR page shows two codes side-by-side for easy scanning:

  • WiFi QR — connects the scanner's phone to the timer network
  • App QR — opens the timer web interface in a browser

Display the QR page on a tablet near the court so players can join without asking for the WiFi password.

QR Settings

Configure the WiFi network name (SSID), password, and encryption type shown in the WiFi QR code.

If the SSID is left blank, the timer uses the network it's currently connected to.

Users & Roles

  • Viewer — can see the timer and events but cannot control anything
  • Operator — can start, stop, pause, and reset the timer
  • Admin — full access including user management, Hello Club settings, and system configuration

Manage operators and change passwords from the user icon in the header (admin only).

Connectivity

  • ● Green dot — connected to the timer
  • ● Red dot — disconnected, auto-reconnecting

The timer auto-reconnects if the connection drops. If it fails after several attempts, refresh the page.

The clock icon shows ✓ when time is synced via NTP, or ✗ if sync failed.
